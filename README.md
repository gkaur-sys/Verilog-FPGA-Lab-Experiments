# Verilog-FPGA-Lab-Experiments
This repository includes complete Verilog implementations, testbenches, waveforms, and FPGA synthesis files for various digital circuits: combinational logic, multiplexers, ripple carry adders, ALUs, SIPO register, universal shift register, ripple counters, and pin-compatible IC modeling (74x72 JK Master-Slave Flip-Flop).
